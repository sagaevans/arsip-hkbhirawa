📋 Import Dus dari Excel

Cara pakai: 1. Pilih data di Excel (boleh dengan header atau tidak)
2. Tekan Ctrl+C untuk copy
3. Klik area teks di bawah lalu tekan Ctrl+V
4. Atur pemetaan kolom, lalu klik Import

📋 Import Dokumen dari Excel

Cara pakai: 1. Di Excel, copy kolom nama dokumen (boleh beserta kolom no)
2. Tekan Ctrl+C untuk copy
3. Klik area teks di bawah lalu tekan Ctrl+V
4. Atur pemetaan kolom, lalu klik Import

📥 Import Dokumen dari Excel (Multi-DUS)

Format kolom Excel yang didukung: Paste data dengan kolom: NO DUS  |  tahun  |  no dokumen
Boleh ada header atau tidak. Sistem otomatis cocokkan NO DUS ke Dus yang ada.
Contoh: 11   2012   15-2060 - 2082
